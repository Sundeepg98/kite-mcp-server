# Tools Reference

Kite MCP provides the following tools for interacting with your Zerodha account.

## Setup Tools

- **login** - Authenticate with your Zerodha account via OAuth

## Account Tools

- **profile** - Get your user profile information
- **margins** - Check available margins and funds
- **holdings** - View your portfolio holdings
- **positions** - View open positions

## Order Tools

- **orders** - List all orders for the day
- **order_history** - Get history of a specific order
- **order_trades** - Get trades for a specific order
- **trades** - List all trades for the day
- **place_order** - Place a new order
- **modify_order** - Modify an existing order
- **cancel_order** - Cancel an open order

## GTT Tools

- **gtt_orders** - List all GTT orders
- **place_gtt_order** - Create a new GTT order
- **modify_gtt_order** - Modify an existing GTT order
- **delete_gtt_order** - Delete a GTT order

## Market Data Tools

- **quotes** - Get full quotes for instruments
- **ltp** - Get last traded price
- **ohlc** - Get OHLC data
- **historical_data** - Get historical candle data
- **instruments_search** - Search for instruments

## Other Tools

- **alerts** - Manage price alerts
- **mf_holdings** - View mutual fund holdings
